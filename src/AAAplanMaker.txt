Main Goals of Refactoring are to include following features:
1. CRUD posts ( very important )
2. Testing ( important )
3. Simplicity ( most important )

1. CRUD posts
1.1. Change how posts are created
1.2. Change how posts are read
1.3. Link to NoSQL
1.4. Make possible to change posts
1.5. Make possible to delete posts

2. Testing
2.1. Setup Testing
2.2. Unit testing
2.3. Integration testing
2.4. E2E testing

3. Simplicity
3.1. No logins
3.2. Enter Info While posting
3.3. Set Default Info
3.4. Responsive page
3.5. Cross Browser Compatability
